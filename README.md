Explanation
-------------
文件在Release里面下载，CN后缀为国服，Global后缀为国际服，B后缀为B服<br>
自V2.1.4版本开始更新双版本资源替换包<br>
后缀带有NoHotUpdates的为无游戏热更新版本<br>
优点是基本整个大版本都可使用，缺点是每次切服时进游戏会重新下载一些游戏资源<br>
而带有游戏热更新的资源包则无需下载游戏资源文件，缺点是文件可能较大且更新较为频繁<br>
